4 Contagem de subconjuntos com soma X

Explicação do problema

Dado um vetor arr de tamanho N e um inteiro X, a tarefa é encontrar a quantidade de subconjuntos de arr que somam X.

Por exemplo, se arr = {1, 2, 3, 3} e X = 6
a saída deverá ser 3, pois temos 3 subconjuntos que somam 6: {1, 2, 3}, {1, 2, 3} e {3, 3}.
Por exemplo, se arr = {1, 1, 1, 1} e X = 4
a saída deverá ser 1, pois só temos 1 subconjunto de arr para que a soma de seus elementos seja 4.

Explicação da solução

estratégia de divisão e conquista

Considerando um array de números naturais.

N = arr.length

e = arr[N - 1]

arr' = arr - {e}

conta_soma(arr, X) = se arr = {}: 0
se arr ≠ {}: se X = e: 1
se X > 0: conta_soma(arr', X) + conta_soma(arr', X - e)

solução recursiva para o problema


conta_soma(arr, X):
    N ← arr.length
    if(N = 0): 
        return 0
    if(N > 0):
        e ← arr[N - 1]
        arr' ← arr[0‥N-2]
        if(X = e):
            return 1
        else if(X > 0):
            return conta_soma(arr', X) + conta_soma(arr', X - e)














subestrutura ótima

com N = arr.length

i \ j 0 1 … X
0 0 0 … 0
1 conta_soma(arr[0‥0], 0) conta_soma(arr[0‥0], 1) … conta_soma(arr[0‥0], X)
2 conta_soma(arr[0‥1], 0) conta_soma(arr[0‥1], 1) … conta_soma(arr[0‥1], X)
… … … …
N conta_soma(arr, 0) conta_soma(arr, 1) … conta_soma(arr, X)

Análise assintótica

conta_soma(arr, X) = O(NX), N = arr.length
